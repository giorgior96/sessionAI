# Skill And Isometric Training

## Sources

- High contextual interference improves retention in motor learning: systematic review and meta-analysis. PMC: https://pmc.ncbi.nlm.nih.gov/articles/PMC11237090/
- The effect of contextual interference on transfer in motor learning: systematic review and meta-analysis. PMC: https://pmc.ncbi.nlm.nih.gov/articles/PMC11349744/

## Practical Principles
- Calisthenics skills are motor tasks and strength tasks. They need specific exposure, high-quality repetitions, and useful feedback.
- Static skills need practice close enough to the target angle/body line to transfer.
- Early learning benefits from stable, repeatable practice; more advanced athletes can use more variation to improve retention and transfer.
- Video feedback is valuable when the performance target is visual: body line, scapular position, pelvic control, elbow lock, and tempo.

## Programming Rules
- Put the key skill or isometric work early in the session, after warm-up and before fatiguing accessories.
- Use short, clean holds or reps. Stop before visible technical collapse.
- Prefer multiple high-quality sets over one maximal ugly attempt.
- Use regressions/bands/leverage changes that let the athlete hit the target shape.
- Add video prompts only for key sets; too much filming can slow the workout and reduce adherence.

## Use In SessionAI
- For goals such as front lever, planche, handstand push-up, and muscle-up, the generated program must include a technical quality criterion.
- The share JSON should include `cameraSuggested: true` only for exercises where feedback changes the next progression decision.
